Under the Hood of Generative AI (LLMs) 7 demos
  • Embedding Space
    Visualize word embeddings in 2D — explore clusters, nearest neighbors, and vector arithmetic like king − man + woman = queen.
  • Single Neuron
    Interactive single-neuron visualization — adjust inputs, weights, and bias, then see how activation functions shape the output.
  • Attention Mechanism
    Explore how the attention mechanism works through an intuitive cocktail-party analogy.
  • Base Model vs Instruct Model
    Compare how a raw base model and an instruction-tuned model respond differently to the same prompts.
  • Illusion of Chat
    Toggle between a polished chat UI and the raw token stream underneath — see how special tokens create the "conversation."
  • Context Filling
    Watch how an LLM's context window gets dynamically filled — system prompts, memories, instructions, skills, and conversation all compete for limited space.
  • Interpreting LLMs
    Explore how researchers peer inside language models — from polysemantic neurons to interpretable features and attribution graphs.
